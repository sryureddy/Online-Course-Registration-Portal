# Online Course Registration Portal

A responsive and user-friendly Online Course Registration Portal developed using HTML5, CSS3, Bootstrap 5, and JavaScript for CVR College of Engineering.

## Live Demo

 **View Live Project**

https://sryureddy.github.io/Online-Course-Registration-Portal/

> Update the link after deploying your project through GitHub Pages.

---

## Project Description

This project provides an online platform where students can:

- View available courses
- Learn course details
- Register for courses
- Login to the portal
- Learn about the institution
- Contact the college

The website is fully responsive and designed using Bootstrap 5 components.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sryureddy/Online-Course-Registration-Portal.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
